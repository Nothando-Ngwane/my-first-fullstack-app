# 📝 Full-Stack Task Manager 
A clean, functional task management application built to demonstrate Full-Stack development principles.

## 🌟 Project Overview
This project is a personal productivity tool that allows users to create, track, and manage daily tasks. It was developed to apply the skills I gained during my **Full-Stack Development Certification**, focusing on connecting a Python backend with a responsive HTML/CSS frontend.

---

## 🚀 Features
- **Task Tracking:** Add and organize daily goals.
- **Responsive Design:** Styled with CSS for a modern look on any device.
- **Backend Logic:** Powered by Python (Flask) to handle data routing.
- **Clean UI:** Simple, intuitive interface for better user experience.

---

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid)
- **Backend:** Python, Flask
- **Version Control:** Git & GitHub

---

## 📈 What I Learned
- How to structure a Full-Stack application.
- The importance of "Separation of Concerns" (keeping UI separate from Logic).
- How to use Git for professional project documentation.

---

## ⚙️ How to Run Locally
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/my-first-fullstack-app.git`
2. Install Flask: `pip install flask`
3. Run the app: `python app.py`
